# Conceptos de Diseño - Krisly Beauty Store

## Concepto 1: Elegancia Minimalista Pastel
**Movimiento de Diseño:** Minimalismo contemporáneo con toques de lujo
**Principios Centrales:**
- Espacios amplios y respiración visual
- Tipografía delicada con jerarquía clara
- Colores pasteles suaves como protagonistas
- Elementos geométricos simples y refinados

**Filosofía de Color:**
- Rosa pastel (#F8D7E6) como color primario
- Blanco roto (#FAFAFA) como fondo
- Crema suave (#FFF5F0) para secciones alternas
- Gris taupe (#B8A8A0) para textos secundarios
- Toques de oro champagne (#E8D5C4) para detalles premium

**Paradigma de Layout:**
- Asimetría elegante con bloques de contenido flotantes
- Galería de productos en grid flexible 3-4 columnas
- Secciones con ancho variable, no centradas uniformemente
- Espacios negativos generosos entre elementos

**Elementos Distintivos:**
- Bordes redondeados suaves (8-12px)
- Líneas divisoras delgadas en tonos pasteles
- Iconografía minimalista en blanco y rosa
- Fotografías con filtro sutil de tonos cálidos

**Filosofía de Interacción:**
- Transiciones suaves de 300-400ms
- Hover effects sutiles con cambio de opacidad
- Botones con efecto de elevación delicada
- Animaciones de entrada lentas y elegantes

**Animaciones:**
- Fade-in de elementos al scroll (200-300ms)
- Hover en productos: escala 1.02 + sombra suave
- Botones: cambio de color + movimiento vertical -2px
- Transiciones de página con fade suave

**Sistema Tipográfico:**
- Display: Playfair Display (serif elegante) para títulos
- Body: Poppins (sans-serif moderno) para textos
- Jerarquía: 48px/36px/24px/18px/16px
- Espaciado de línea: 1.6 para cuerpo, 1.2 para títulos

## Concepto 2: Belleza Romántica y Artesanal
**Movimiento de Diseño:** Cottagecore moderno + Maximismo suave
**Principios Centrales:**
- Texturas y patrones visuales
- Abundancia de detalles decorativos
- Paleta cálida y envolvente
- Sensación de handmade y personalización

**Filosofía de Color:**
- Rosa magenta suave (#F5C2D6) como protagonista
- Fondo crema con textura (#FFFBF7)
- Púrpura lavanda (#E8D5E8) para acentos
- Terracota suave (#E8B4A8) para detalles
- Verde salvia (#C8D8D0) para contraste natural

**Paradigma de Layout:**
- Secciones con marcos decorativos
- Disposición orgánica con elementos superpuestos
- Galerías con tamaños variables (masonry)
- Bordes y marcos ornamentales sutiles

**Elementos Distintivos:**
- Patrones florales sutiles de fondo
- Bordes decorativos en secciones clave
- Ilustraciones botánicas minimalistas
- Tarjetas con sombras suaves y bordes decorados

**Filosofía de Interacción:**
- Movimientos más expresivos y lúdicos
- Efectos de parallax suave en scroll
- Animaciones de "bloom" en hover
- Detalles micro-interactivos

**Animaciones:**
- Elementos que "flotan" suavemente (transform: translateY)
- Hover: rotación -1° + escala 1.03 + sombra cálida
- Transiciones de página con efecto de cortina
- Animación de entrada con bounce suave

**Sistema Tipográfico:**
- Display: Cormorant Garamond (serif clásico) para títulos
- Body: Quicksand (sans-serif suave) para textos
- Jerarquía: 52px/40px/28px/20px/16px
- Espaciado de línea: 1.7 para cuerpo

## Concepto 3: Modernidad Limpia y Accesible
**Movimiento de Diseño:** Design System contemporáneo + Neomorfismo suave
**Principios Centrales:**
- Claridad y funcionalidad ante todo
- Contraste accesible pero estético
- Componentes reutilizables y consistentes
- Enfoque en experiencia del usuario

**Filosofía de Color:**
- Rosa coral suave (#F5A9B8) como primario
- Blanco puro (#FFFFFF) para máxima claridad
- Gris neutro (#E8E8E8) para separaciones
- Rosa oscuro (#D4647F) para textos y CTA
- Azul pastel (#D4E8F0) para información

**Paradigma de Layout:**
- Grid simétrico pero con variaciones
- Secciones de ancho completo con contenedor centrado
- Tarjetas uniformes con espaciado consistente
- Navegación clara y predecible

**Elementos Distintivos:**
- Botones con estados claros (normal, hover, active)
- Tarjetas con sombra sutil y borde delicado
- Iconografía limpia y coherente
- Uso estratégico de espacios en blanco

**Filosofía de Interacción:**
- Feedback inmediato y claro
- Transiciones predecibles y rápidas
- Estados visuales bien definidos
- Accesibilidad como prioridad

**Animaciones:**
- Transiciones rápidas (150-250ms)
- Hover: cambio de color + sombra
- Botones: pulso suave al hacer clic
- Carga de contenido con skeleton screens

**Sistema Tipográfico:**
- Display: DM Sans Bold para títulos
- Body: DM Sans Regular para textos
- Jerarquía: 44px/32px/24px/18px/16px
- Espaciado de línea: 1.5 para cuerpo, 1.3 para títulos

---

## Decisión Final
Se selecciona el **Concepto 1: Elegancia Minimalista Pastel** por su capacidad de reflejar la sofisticación y belleza de la marca Krisly Ramirez, manteniendo una accesibilidad visual clara y un aspecto premium que atrae al público objetivo.
